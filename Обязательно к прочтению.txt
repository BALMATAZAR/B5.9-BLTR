Основной скрипт - example.py, с помощью его запускаются остальные скрипты.
Допустим, чтобы запустить скрипт timer.py, необходимо ввести в консоли windows следующее:

python example.py timer.py 

Можно прогнать скрипт некоторое количество раз, просто добавив цифру в конце, например:

python example.py timer.py 5

Далее выполнится скрипт, который укажет имя скрипта(который запустили), количество повторений выполнения 
скрипта, а так же общее и среднее время выполнения

Описание файлов в домашнем задании:

- example.py - скрипт для запуска из командной строки.

- timer - скрипт для создания декоратора, позволяет импортировать декоратор в свой код как отельный модуль.

- timer_class - скрипт для использования класса в контекстном менеджере.
